
# Generic Platform UI


## Project Info

This is a generic document extraction and validation UI built with React, Vite, TypeScript, shadcn-ui, and Tailwind CSS.


## How to run and develop

Clone the repository and use your preferred IDE. You need Node.js & npm installed ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm install

# Step 4: Start the development server with auto-reloading and instant preview.
npm run dev
```


## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS


## Deployment

You can deploy this project using any static hosting provider (e.g., Vercel, Netlify, GitHub Pages) after building with:

```sh
npm run build
```

The output will be in the `dist` folder.
